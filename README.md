# 📊 Investment Banking Analysis using Python

## Overview
This project focuses on analyzing investor behavior, investment preferences, savings objectives, investment duration, and expected returns using Python. The analysis was performed using data cleaning, exploratory data analysis (EDA), statistical techniques, and data visualization to derive meaningful business insights.

---

## Objective
The goal of this project is to understand investment patterns and identify key factors influencing investment decisions through data-driven analysis.

- Identified the most preferred investment avenues among investors.
- Analyzed major savings objectives and investment goals.
- Studied investment duration patterns.
- Evaluated common sources of investment information.
- Explored relationships between investment-related variables using correlation analysis.
